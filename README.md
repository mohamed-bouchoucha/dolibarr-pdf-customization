# Dolibarr PDF Customization Module

## 📌 Project Overview
Customization of Dolibarr PDF templates (Invoices, Orders, Proposals)
based on business and branding requirements.

## ⚙️ Tech Stack
- PHP
- TCPDF
- Dolibarr 23
- Git

## 📂 Project Structure
/custom/pdfcustom → custom PDF templates

## 🚀 Features
- Custom invoice layout
- Branding (logo, colors)
- Dynamic business fields
- Conditional display logic
